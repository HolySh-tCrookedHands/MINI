# MINI

`MINI — минимум зависимостей, минимум рисков, максимум общения.`

Основная концепция приложения завязана на том, что любой желающий может запустить сервер у себя дома.
Данный аспект позволяет децентрализовать проект, а также расширить комьюнити до масштабов космоса, с учётом того, что в комплекте с сервером идёт VPN-туннель, который связывает клиент и сервер.  
  
Основные настройки содержат в себе функционал, полезный как для бизнеса (в виде опции `«ручная регистрация»`), так и для маленьких компаний друзей с авторегистрацией через интерфейс клиента.  
  
**Panic Button:**
Нажмите одну кнопку, и мессенджер мгновенно сотрет ключи шифрования, перезапишет базу данных случайным мусором и разорвет VPN-туннель.  
На устройстве не останется ни одного следа вашего общения.
